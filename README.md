# dunder-mifflin [![Build Status](https://travis-ci.org/GameDevFox/dunder-mifflin.svg?branch=master)](https://travis-ci.org/GameDevFox/dunder-mifflin)
A RESTful API for the new Dunder Mifflin Infinity website

## Running

```
$ ./gradlew bootRun
```

## Testing

```
$ ./gradlew check
```
